# SEO Phase 2 Recommendations (Requires UI Approval)

The following recommendations have been identified during the Phase 2 SEO Audit. **These require explicit UI approval before implementation because they will visually alter the website.**

## 1. /from/ranchi Dedicated Landing Page (P3)
Currently, `/from/[city]` is generated dynamically but is set to NOINDEX to prevent thin content penalties.
**Recommendation**: For `/from/ranchi`, design a unique landing page template that includes:
- A unique localized introduction ("Kamakhya Yatra: Your Trusted Travel Agency in Ranchi").
- Genuine connectivity information (Train routes from Ranchi, Flight connectivity from Birsa Munda Airport).
- A localized FAQ section addressing Ranchi departures.
**Status**: REQUIRES UI APPROVAL. (Do not index until content is populated).

## 2. On-Page Tour FAQs (P3)
Adding a dedicated FAQ accordion to the bottom of individual tour pages (e.g., "Char Dham Yatra").
**Recommendation**: Implement a sleek, collapsible FAQ block below the itinerary.
**SEO Benefit**: Allows injection of long-tail keywords and makes the page eligible for `FAQPage` rich snippets in Google Search.
**Status**: REQUIRES UI APPROVAL.

## 3. Blog Layout & Content Publishing (P2)
The `/blog` route exists but has no individual articles yet.
**Recommendation**: Design and approve a `/blog/[slug]` reading layout. Once approved, begin publishing the articles outlined in `SEO_CONTENT_PLAN.md`.
**Status**: REQUIRES UI APPROVAL for the article layout design.

## 4. Internal Link "Quick Links" Footer (P2)
Currently, internal linking relies on in-text contextual links.
**Recommendation**: Add a visually clean "Popular Tours from Ranchi" block in the footer or pre-footer to distribute link equity to top packages.
**Status**: REQUIRES UI APPROVAL.
